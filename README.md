# EMODZ: Proprietary Emotion Analysis AI

EMODZ is a high-performance **Emotion Analysis AI** specifically engineered for the linguistic complexities of the Algerian digital landscape. Unlike generic sentiment tools, EMODZ is powered by a proprietary **DziriBERT (Emotion Edition)** model, custom-trained and optimized in-house.

**[🌐 Access EMODZ Dashboard (DZMO)](http://209.38.249.154:8501)**

---

## 🧠 Proprietary AI Architecture

### 1. Custom-Trained DziriBERT
The heart of EMODZ is a fine-tuned **DziriBERT** transformer, developed using the proprietary architecture documented in our [Training Notebook](Working_V1_1_.ipynb).
- **Linguistic Mastery**: Native understanding of **Darija**, **Algerian Franco-Arabe**, and **Modern Standard Arabic**.
- **11-Emotion Deep Scope**: The AI is trained to detect nuanced emotional states including:
  - *Satisfaction*, *Frustration*, *Urgency*, *Sarcasm*, *Inquiry*, *Disappointment*, *Gratitude*, *Anger*, *Confusion*, *Suggestion / Feedback*, and *Humor / Joking*.
- **Layer-Preserved Intelligence**: Our training process utilizes advanced layer-freezing to retain foundational "Algerian Knowledge" while optimizing for emotion-specific nuances.

### 2. 100% Local & API-Free
EMODZ operates as a fully autonomous, offline AI system. 
- **Zero External APIs**: We do NOT use OpenAI, Claude, or any third-party APIs. All processing is 100% local.
- **Privacy First**: Sensitive social data and insights never leave your secure environment.
- **Local Executive Briefs**: Real-time summaries are generated via a locally-hosted **Gemma 3:4b** via Ollama, ensuring zero recurring costs and total data sovereignty.

### 3. High-Yield AI Scraper
Feeds the emotion engine with high-fidelity raw data bypassing traditional bot-detection walls.
- **Deep Thread Expansion**: Proactively captures nested comments to provide the AI with context-rich data.
- **Anonymous Analysis**: Strips metadata to focus strictly on pure emotional intelligence.

## 🛠️ Quick Start (Zero-Touch Setup)

**1. Clone the repository**
Ensure [Git LFS](https://git-lfs.com/) is installed to download the proprietary model weights (`saved_model/`).
```bash
git lfs install
git clone <your-repository-url>
cd EMODZ
```

**2. Launch the AI**
Double-click **`Start_EMODZ.bat`**. The system will automatically provision the local AI environment and ready the Emotion engine.
