## 🛠️ Quick Start (Local Setup)

The EMODZ platform is designed for zero-touch deployment on Windows.

**1. Clone the repository**
Ensure [Git LFS](https://git-lfs.com/) is installed to handle the 400MB+ AI model weights.
```bash
git lfs install
git clone <your-repository-url>
cd EMODZ
```

**2. Launch the Platform**
Double-click **`Start_EMODZ.bat`**. The system will automatically:
- Provision a Python Virtual Environment.
- Pull the Gemma 3:4b model via Ollama.
- Setup Playwright/Chromium dependencies.
- Open the dashboard in your default browser.
